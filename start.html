Let's get started with Giblog

Giblog is a tool for creating websites and blogs that can be managed with git. In Japan, it is also called CMS. Managing websites created with Giblog with Git makes it easier to manage multiple websites.

12 Reasons to Use Giblog for Website Creation

  • Manage all pages with git
  • Advanced SEO measures
  • Responsive design for smartphones
  • Contact form support
  • Fast rebuild and automation of rebuild
  • Strong security
  • Articles can be searched and replaced with Linux commands
  • Supports multiple OS such as Windows, Mac OS, Linux
  • You can customize your site freely with Perl
  • High performance web page
  • Publish your site completely free of charge using Github Pages
  • Forum user support
