ブログのトップページを自分で作成するには

new_blogコマンドで作成したブログは、1週間分の記事を掲載したトップページが自動的に生成されます。ブログのトップページを静的なHTMLのページにする方法を解説します。

ビルドスクリプトを開く

まず最初に、サイトをビルドしているPerlスクリプトを開きます。 

lib/Giblog/Command/build.pm

トップページの自動生成の処理をコメントアウト

次のコードを探してください。 

  # Create index page
  $self->create_index;

この部分をコメントアウトします。Perlのコメントは「#」です。 

  # Create index page
  # $self->create_index;

トップページのHTMLを配置

最後に、自分でトップページを作成します。以下のファイルを作成してください。 

templates/index.html

内容はトップページの内容。 

<h2>わたしのサイト</h2>

わたしのサイトです。
