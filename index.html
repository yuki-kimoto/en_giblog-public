Introduction of Giblog

Giblog is a tool for creating sites and blogs that can be managed by Git.

Managing websites created with Giblog with Git makes it easier to manage multiple websites.

It solves "I want to replace only here", "It's hard to back up", and "I want to edit in the local environment".
